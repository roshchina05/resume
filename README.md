## Состав

### 01. Разработка корпоративной информационной системы отдела главного механика на НПЗ

Автоматизация управления ремонтами оборудования для нефтегазового предприятия.

- Обследование процессов AS IS, моделирование IDEF0, переход к TO BE
- Технико-экономическое обоснование: потери 13,5 млн руб/год, окупаемость 4 месяца
- C-требования и D-требования для 6 АРМ по ГОСТ 34
- Проектирование реляционной БД (прототип)
- HTML/CSS-прототип интерфейса
- Тендерная документация по 223-ФЗ

**Папка:** razrabotka_kis

### 02. Жизненный цикл контейнера безопасности (модуль сквозного шифрования трафика)

Защита данных, передаваемых по Wi-Fi между терминалами сбора данных и сервером.

- Техническое задание по ГОСТ 19.201
- Функциональные требования: аутентификация, обмен ключами, AES-256-GCM
- Комплект тестовой документации по IEEE 829 и ГОСТ Р 56920
- Модульные тесты на pytest
- Ручное приёмочное тестирование, протокол испытаний

**Папка:** security_container

### 03. Анализ требований к платформе онлайн-подготовки к ЕГЭ

Система дистанционного обучения (онлайн школа): вебинары, домашние задания, обратная связь от преподавателей, отслеживание прогресса.

- Диаграмма IDEF0 процесса «Управление подготовкой к ЕГЭ с использованием СДО»
- Диаграммы AS IS и TO BE
- Декомпозиция подпроцессов в BPMN: изучение контента, отправка заданий, корректировка учебного плана, подготовка ответа
- DMN-модель выбора контента (дерево решений и таблица)
- 4 Use Cases с полным описанием потоков и исключений
- Каталог бизнес-правил
- Бизнес-требования, сгруппированные по категориям

**Папка:** edtech_ege

### 04. Проектирование базы данных для платформы онлайн-фитнеса

Платформа с программами тренировок и отслеживанием прогресса (по типу fitstart).

- ER-диаграмма (8 сущностей)
- Модель ключей, полноатрибутная модель
- Нормализация до 3NF

**Папка:** bd_fitness

### 05. Планирование ИТ-проекта Киберняня

Приложение родительского контроля для защиты детей в интернете.

- Декомпозиция MVP на 6 модулей
- WBS, диаграмма Ганта на 7 месяцев
- Матрица ответственности RACI
- Расчёт бюджета и ресурсов
- План коммуникаций и реестр рисков
- Расчёт резервного фонда

**Папка:** kiber_babysitter

### 06. Эконометрические исследования

**Исследование 1: факторы, влияющие на ставки по жилищным кредитам**
- Регрессионный анализ в Excel
- Тест Чоу, тест Уайта, критерий Дарбина-Уотсона

**Исследование 2: моделирование и прогнозирование цен на нефть WTI**
- Регрессионный анализ в Excel
- Модели AR(1), AR(2), трендовая модель
- h-критерий Дарбина, проверка стационарности остатков
- Сравнительный прогноз трёх моделей

**Папка:** econometrics
