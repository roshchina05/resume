Учебные проекты, выполненные в ходе обучения. Охватывают полный цикл: от обследования и написания ТЗ до проектирования баз данных, прототипирования интерфейсов и тестирования.

## Структура

### 01. КИС отдела главного механика

Автоматизация управления ремонтами оборудования для нефтегазового предприятия.

- Обследование процессов AS IS, моделирование IDEF0, переход к TO BE
- Технико-экономическое обоснование: потери 13,5 млн руб/год, окупаемость 4 месяца
- C-требования и D-требования для 6 АРМ по ГОСТ 34
- Проектирование реляционной БД
- HTML/CSS-прототип интерфейса
- Тендерная документация по 223-ФЗ

**Файлы:** ТЭО, C/D-требования, SQL-скрипты, прототип, тендер

### 02. ТЗ на модуль сквозного шифрования трафика

Защита данных, передаваемых по Wi-Fi между терминалами сбора данных и сервером.

**Что сделано:**
- Техническое задание по ГОСТ 19.201
- Функциональные требования: аутентификация, обмен ключами, AES-256-GCM

**Файлы:** ТЗ, спецификация требований

### 03. EdTech-платформа для подготовки к ЕГЭ

Система дистанционного обучения с вебинарами, заданиями и отслеживанием прогресса.

**Что сделано:**
- SWOT-анализ и PESTLE-анализ рынка
- 4 Use Cases с полным описанием потоков и исключений
- Каталог из 15+ бизнес-правил
- Матрица трассировки требований

**Файлы:** Use Cases, бизнес-правила, матрица трассировки

### 04. Проектирование БД для онлайн-фитнеса

База данных для платформы с программами тренировок и отслеживанием прогресса.

**Что сделано:**
- ER-диаграмма на 8 сущностей
- Модель ключей, полноатрибутная модель
- Разрешение связи M:N через дополнительную сущность
- Нормализация до 3NF

**Файлы:** ER-диаграмма, модель ключей, пояснительная записка

### 05. Планирование ИТ-проекта StepChef

Мобильное приложение с пошаговыми рецептами и интеграцией доставки продуктов ВкусВилл.

**Что сделано:**
- Продуктовая и функциональная WBS
- Матрица ответственности RACI
- Диаграмма Ганта, расчёт бюджета и ресурсов
- План коммуникаций и реестр рисков
- Ведение задач в Yandex Tracker

**Файлы:** WBS, RACI, диаграмма Ганта, план коммуникаций

### 06. Планирование ИТ-проекта Киберняня

Приложение родительского контроля для защиты детей в интернете.

**Что сделано:**
- Декомпозиция MVP на 6 модулей
- WBS, диаграмма Ганта на 7 месяцев
- Матрица рисков, расчёт резервного фонда

**Файлы:** WBS, матрица рисков

### 07. Экспертный анализ рисков ИИ-ассистента Яндекс Лавки

Оценка рисков внедрения ИИ-консультанта в продуктовый ретейл.

**Что сделано:**
- Идентификация 24 рисков
- Количественная оценка с учётом компетентности экспертов
- Метод «галстук-бабочка» для критических рисков

**Файлы:** анализ рисков, матрица, галстук-бабочка

### 08. Тестирование криптографического модуля

Проверка корректности шифрования AES-256-GCM и устойчивости к атаке с чужим ключом.

**Что сделано:**
- Комплект тестовой документации по IEEE 829 и ГОСТ Р 56920
- Модульные тесты на pytest
- Ручное приёмочное тестирование, протокол испытаний

**Файлы:** план тестирования, спецификация тестов, протокол

### 09. Эконометрика: ипотечные ставки в РФ

Исследование факторов, влияющих на ставки по жилищным кредитам.

**Что сделано:**
- Многофакторная регрессия (R² = 0,85)
- Тест Чоу, тест Уайта, критерий Дарбина-Уотсона

**Файлы:** отчёт, расчёты

### 10. Эконометрика: цены на нефть WTI

Моделирование и прогнозирование спотовых цен на нефть.

**Что сделано:**
- Модели AR(1), AR(2), трендовая модель
- h-критерий Дарбина, проверка стационарности остатков
- Сравнительный прогноз трёх моделей

**Файлы:** отчёт, расчёты
